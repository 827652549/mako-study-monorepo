# Claude Code 演示项目

本项目用于测试和探索 Claude Code 的各种特性和行为。

## 项目目标

这是一个用于实验 Claude Code 能力的测试场，包括：

- **代码生成** - 测试 AI 辅助代码编写和补全
- **代码理解** - 分析 Claude Code 如何读取和解释现有代码
- **重构** - 测试代码转换和优化建议
- **多文件操作** - 跨多个文件和目录工作
- **Git 集成** - 提交创建、分支管理和仓库操作
- **任务规划** - 测试待办事项列表和任务管理功能
- **上下文管理** - 理解 Claude Code 如何在对话中维护上下文
- **工具使用** - 探索各种工具（Read、Write、Edit、Bash 等）
- **错误处理** - Claude Code 如何响应和修复错误
- **命令执行** - 测试 bash 命令和脚本执行

## 测试领域

### 1. 代码生成
- 从零生成新文件
- 创建样板代码
- 实现特定算法或模式

### 2. 代码分析
- 解释现有代码
- 查找 bug 和潜在问题
- 提出改进建议

### 3. 文件操作
- 创建、读取、更新、删除文件
- 处理不同文件类型（JS、TS、JSON、MD 等）
- 处理大文件和目录

### 4. Git 工作流
- 暂存和提交更改
- 创建描述性提交消息
- 管理分支
- 推送到远程仓库

### 5. 开发工具
- 包管理器操作（npm、pnpm、yarn）
- 构建和测试命令
- 代码检查和格式化

### 6. 复杂任务
- 多步骤操作
- 任务规划和跟踪
- 错误恢复和重试逻辑

## 说明

这是一个实验性项目。代码质量和模式可能会有所不同，因为我们会测试不同的方法和特性。

## 开始使用

此目录将包含各种测试文件和实验。每个测试用例都会在我们探索 Claude Code 能力时进行记录。

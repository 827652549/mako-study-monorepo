🎬 Web 播放器学习 Demo

1️⃣ Media Source Extensions (MSE) Demo

MSE 是什么？ 让浏览器动态接收和处理视频片段，不需要完整的视频文件。常用于 HLS/DASH 直播和点播。
// MSE 核心 API const mediaSource = new MediaSource(); const sourceBuffer = mediaSource.addSourceBuffer('video/mp4; codecs="avc1.42E01E"'); sourceBuffer.appendBuffer(arrayBuffer); // 添加视频片段

dash.js 是什么？ 开源的 DASH (Dynamic Adaptive Streaming) 播放器，自动切换码率以适应网络条件。

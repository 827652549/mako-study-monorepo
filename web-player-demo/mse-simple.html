📺 MSE (Media Source Extensions) 实际工作演示

演示说明： 这个例子展示 MSE 如何动态加载和播放视频片段。
使用公开的 DASH 流，提取片段数据，通过 MediaSource API 播放。
状态： 就绪
工作原理：
  1. MediaSource.isTypeSupported() - 检查格式支持
  2. video.src = URL.createObjectURL(mediaSource) - 绑定媒体源
  3. mediaSource.addSourceBuffer() - 创建缓冲区
  4. sourceBuffer.appendBuffer() - 添加视频数据
  5. mediaSource.endOfStream() - 标记流结束
💡 提示： 这个演示使用公开 DASH 流数据。真实应用中，你需要：
1. 获取 MPD manifest 文件
2. 解析视频的初始化段
3. 分段获取和追加媒体片段
4. 根据网络条件动态选择清晰度
